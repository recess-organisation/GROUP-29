# UGScholar

Adaptive learning platform for Uganda — accessible via USSD/SMS on any MTN or Airtel line, no smartphone needed.

## Features
- USSD/SMS-based access (no internet required)
- PIN authentication with NITA-U compliance
- Age-based curriculum routing (P.1–P.7, O-Level, A-Level, Adult Literacy)
- Local language support (Luganda, Runyankole, Acholi, Ateso, English)
- Learning gap detection via diagnostic quizzes
- Adaptive micro-lessons delivered via SMS
- Gamification: points, badges, streaks, district leaderboards, airtime prizes

## Tech Stack
- **Backend:** Node.js + Express
- **Database:** TBD
- **SMS/USSD:** Africa's Talking API
- **Frontend:** Web dashboard (optional)
